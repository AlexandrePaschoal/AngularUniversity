AngularUniversity

Explore universidades ao redor do mundo

Sobre o projeto AngularUniversity é uma aplicação desenvolvida com Angular e Ionic que permite explorar universidades ao redor do mundo utilizando dados provenientes de uma API pública. O projeto oferece funcionalidades como pesquisa por país, visualização de resultados, sistema de favoritos, histórico de pesquisas e uma interface moderna e responsiva. Seu objetivo é demonstrar a aplicação de conceitos de desenvolvimento frontend, consumo de APIs REST, gerenciamento de estado e construção de experiências mobile utilizando tecnologias web modernas. Tecnologias
Angular 20
Ionic
